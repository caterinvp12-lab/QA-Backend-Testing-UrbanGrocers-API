# API Testing: Urban Grocers Backend

## Resumen del Proyecto
Este proyecto consistió en la verificación y validación de los nuevos requisitos del backend para la plataforma Urban Grocers, específicamente para las funcionalidades de gestión de Kits y logística de Servicios de Entrega.

## Alcance de las Pruebas
* Análisis de documentación técnica en Apidoc.
* Diseño de casos de prueba para validación de métodos HTTP y parámetros de entrada.
* Ejecución de pruebas funcionales de API mediante Postman.
* Verificación de códigos de respuesta, esquemas JSON y tiempos de carga.
* Documentación y reporte de errores técnicos en Jira.

## Herramientas
* Postman (Pruebas de API)
* Jira Software (Gestión de Defectos)
* Apidoc (Documentación Técnica)

## Documentación Adjunta
En este repositorio se encuentran los reportes de errores generados, la lista de comprobación de los requisitos analizados y las capturas de pantalla de las validaciones exitosas en el servidor.
